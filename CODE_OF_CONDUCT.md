# Code of Conduct

## The short version

Be decent. This project exists because someone needed it. The people using it often have ADHD, chronic illness, or other things making life harder than it needs to be. Keep that in mind.

## What this means in practice

- Assume good faith. Someone asking a basic question is not wasting your time.
- Be direct, not cruel. Honest feedback is fine. Contempt is not.
- No harassment, no slurs, no pile-ons.
- Stay on topic. This is a productivity tool, not a debate floor.

## Scope

This applies in Discussions, Issues, pull requests, and anywhere you interact with this project or its community.

## Enforcement

This is a one-person project. Violations will result in removal from the discussion. Serious or repeated violations will result in a ban. There is no appeals committee.

## Reporting

If something happens that should not, open a General discussion marked private, or contact via GitHub. It will be handled quietly and without drama.

---

This code of conduct is intentionally plain. The spirit of it matters more than the letter.
